Rust Path Planning
BFS, Dijkstra, A*

Project Overview

This Rust-based project implements classic path planning algorithms—BFS, Dijkstra, and A*—with real-time visualizations. By leveraging Rust’s performance and safety features, the system uses optimized data structures such as BinaryHeap and HashMap to ensure efficient computation. The plotters library is used to generate dynamic GIFs, clearly illustrating each algorithm’s step-by-step process, including explored nodes, obstacles, and the final path.

Real-World Applications

These algorithms have significant applications in robotics and autonomous systems, including:

  • Robot Navigation: Enabling robots to move safely through structured environments.
  • Autonomous Vehicles: Planning efficient routes while avoiding obstacles in dynamic settings.
  • Game AI: Pathfinding for characters in grid-based maps or virtual worlds.

This project serves as a robust foundation for exploring and scaling solutions to real-world challenges, with room for extending to advanced algorithms like RRT* or integrating real-world sensor data for adaptive planning.
